# Sales Manager Agent - HEARTBEAT.md

**Frequency**: Daily (pipeline checks), Weekly (Monday 10am all-hands, Wednesday 1:1s)

---

## Daily Heartbeat (Morning)

```
1. Check overnight alerts:
   - Any new leads/prospects?
   - Customer escalations?
   - Deal status changes?

2. Pipeline review:
   - Which prospects move to next stage this week?
   - Any deals at risk of falling through?
   - Blockers from Eng (delivery feasibility)?

3. Actions:
   - Outreach to top 3 prospects
   - Follow-up on negotiations
   - Coordinate with Eng (confirm feasibility before promising timeline)

4. End of day:
   - Update pipeline in Paperclip
   - MRR tracking (current vs target)
   - Sales Rep task delegation
```

---

## Weekly All-Hands (Monday 10am, part of CEO sync)

```
Report to: CEO + all Managers

Contents (5 min):
  - Pipeline: # prospects, # in negotiation, $ amount
  - MRR: Current vs target, $ gap
  - Deals closed this week: #
  - Blockers: Eng feasibility? Contract issues? Pricing questions?
  - Hiring: Need more sales reps?

Actions:
  - Escalate any deals >$50k to CEO
  - Flag Eng blockers → coordinate with Tech Lead
  - Document decisions in Paperclip
```

---

## Weekly 1:1s with Sales Reps (Wednesday, 30 min each)

```
Attendees: Sales Manager, Sales Rep 1/2

Contents:
  - Pipeline status: # prospects, conversion rate
  - Top deal this week: What's the story? Any blockers?
  - Coaching: Sales playbook refresh? Objection handling?
  - Metrics: Calls made, demos done, proposals sent

Actions:
  - Adjust strategy if needed
  - Celebrate wins
  - Flag customer escalations
```
