Full Eval Flow
Agent runs task → logs to Postgres
→ Prometheus scrapes → Grafana displays

Developer makes change → Eval Engine triggers
→ pulls baseline from Postgres
→ runs DeepEval sidecar
→ compares 5 dimensions before vs after
→ produces change report
→ promote or rollback

Mentor reads skill_level delta + trust_score
→ triggers graduation (trainee → autonomous)
→ or coaching when skill regresses
